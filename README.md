**RummyStats** es una aplicación web progresiva diseñada para gestionar las puntuaciones de tus partidas de Rummy de forma ágil, visual y sin necesidad de papel y lápiz.

---

## ✨ Características Principales

* **Gestión Multijugador:** Añade tantos jugadores como necesites en tiempo real.
* **Modificadores Especiales:**
    * ⭐ **Estrella (+50):** Suma automáticamente 50 puntos al valor introducido.
    * ⚡ **Multiplicador (x2):** Duplica el valor del input (se aplica después del bono de estrella).
* **Historial Detallado:** Consulta cada movimiento individual de cada jugador mediante un modal de historial.
* **Estadísticas Avanzadas:** Página dedicada con gráficas de evolución temporal (Chart.js) para ver quién lidera la partida ronda a ronda.
* **Persistencia Local:** Los datos no se pierden al cerrar el navegador gracias al uso de `localStorage`.
* **Diseño Responsive:** Optimizado para móviles, tablets y escritorio.

## 🚀 Tecnologías Utilizadas

* **HTML5 & CSS3:** Estructura y estilos personalizados con efectos de "Glassmorphism".
* **Bootstrap 5:** Framework para el diseño responsivo y componentes (modales, dropdowns).
* **JavaScript (Vanilla):** Lógica del juego y manipulación del DOM.
* **Chart.js:** Renderizado de gráficas dinámicas para las estadísticas.
* **Bootstrap Icons:** Set de iconos vectoriales.
